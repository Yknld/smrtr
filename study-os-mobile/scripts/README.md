# Scripts

This directory will contain build, deployment, and automation scripts for the mobile application.

## Planned Scripts

- **setup.sh** - Initialize development environment
- **build-ios.sh** - Build iOS app bundle
- **build-android.sh** - Build Android APK/AAB
- **test.sh** - Run test suite
- **lint.sh** - Run linters and formatters
- **deploy.sh** - Deploy to app stores (future)
